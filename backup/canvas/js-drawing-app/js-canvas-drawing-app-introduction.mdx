- Zen이 왜 강의를 수강하면서 학습을 하는지 이해한다.
- 본 섹션의 목적을 이해한다.

## 🤔 난 왜 이 강의를 통해 공부하게 되었는가?

[바닐라 JS로 그림 앱 만들기](https://nomadcoders.co/javascript-for-beginners-2) 강의를 수강하면서 공부한 내용을 정리한 섹션이다.<br/>

네이버 부스트캠프 그룹 프로젝트를 진행하면서, 지도 위에 캔버스 레이어를 놓고 조작하는 기능을 구현해야 했다.<br/>

이를 위해서는 `Canvas`에 대한 이해가 필수적이었는데, 적당한 참고자료를 찾지 못했다.<br/>

마침 굉장히 짧은 분량의 노마드코더님의 강의가 있었고, 이를 통해 `Canvas`에 대한 기본적인 이해를 하기 위해서 강의를 수강하게 되었다. <br/>

## 🚀 난 무엇을 얻어가고자 하는가?

이 강의를 통해 `Canvas`에 대한 기본적인 이해를 얻고, 이를 바탕으로 지도 위에 캔버스 레이어를 놓고 조작하는 기능을 구현할 수 있도록 학습하고자 한다.<br/>

동시에, `Canvas` 자체에 대해서 조금 더 친해져보고자 한다.<br/>

## 🤔 캔버스란 무엇인가?

`캔버스(Canvas)`란, 웹 페이지에 그래픽을 그리기 위한 HTML 요소이다.<br/>

```html
<canvas></canvas>
```

이 `canvas html element`는 `canvas API`의 창이다.<br/>

`canvas API`는 `Javascript`로 그래픽을 그릴 수 있게 해주는 `API`이다.<br/>

`WebGL API`로 2D, 3D 그래픽을 그릴 수 있게 해준다. <br/>

여기서는 2D에 대해서만 다룰 예정이다. <br/>

재밌게도, 캔버스는 `JS`로 그래픽을 다루는 여러 요소를 제공한다. 그에 따라서, `JS` 코드를 이용해서 인터렉티브한 그래픽을 만들 수 있다.<br/>
