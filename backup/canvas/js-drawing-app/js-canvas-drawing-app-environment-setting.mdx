## 🎯 이 문서를 읽고 난 후의 상태

- 바닐라 JS로 그립 앱 만들기 위한 개발 환경 설정을 할 줄 안다.

## 📝 개발 환경 설정

아무 디렉토리나 만들고, `index.html`, `app.js`, `styles.css` 를 만든다. <br/>

그리고 다음과 같은 코드를 작성한다.<br/>

```html title="index.html"
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Meme Maker</title>
	<link rel="stylesheet" href="style.css">
</head>
<body>
	<canvas></canvas>
	<script src="app.js"></script>
</body>
</html>
```

```css title="styles.js"
body {
	background-color: tomato;
}
```

`app.js`는 아직 작성을 하지 않았기에 이정도만 작성해준다. <br/>
