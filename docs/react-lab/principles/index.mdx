## 리액트를 깊게 이해하기 위한 시도

오늘날의 프론트앤드 개발에서, 리액트가 차지하는 비중이나 중요성은 굉장히 크게 느껴집니다.

리액트를 배우면서, 리액트를 사용하는 방법에 대해서만 알았지, 비중이나 중요성에 비해서 깊게 이해하지는 못했던 것 같습니다.

리액트가 무슨 문제를 해결하기 위해 나왔는지, 어떻게 문제 해결에 접근하고 있는지 등을 이해하기 위해서 탐구하고자 이 카테고리를 만들게 되었습니다.

본 문서에서는 리액트란 무엇이며, 이게 어떻게 동작하고, 어떤 문제를 해결하기 위해 나왔는지를 다루고자 합니다.
