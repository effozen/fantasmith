## 🔬 React 실험실 🔬

> 리액트를 정말 잘 다루고 싶다. <br/>
> 어디가서 "나 리액트 잘해. 그리고 잘 알아." 라고 말하고 싶다. <br/>

시작은 위와 같은 마음에서 시작이 되었습니다. <br/>

리액트를 정말 잘 다루고 싶은 마음.<br/>

제가 개발을 진행하면서 제일 재밌다고 느낀 것은 어떤 에러나, 개발 현상, 그리고 마주하는 문제를 제가 아는 지식으로 논리 정연하게 찾아가면서 풀었을 때 입니다.<br/>

<ul>
  <li>문제 상황 파악</li>
  <li>원인에 대한 기술적인 분석</li>
  <li>분석한 내용을 바탕으로 해결 방법 추론</li>
  <li>추론한 내용을 직접 구현해보면서 해결</li>
</ul>

이 과정을 거칠 때가 정말 재밌고, 행복한 것 같습니다.<br/>

어설프게 `보안`과 `백앤드`를 맛보기도 했습니다. 학부생 수준이긴 하나, 이때 배운 지식으로 제가 직면했던 몇 가지 문제들을 풀 수 있었습니다. 대표적으로 지금 보고 계신 블로그가 그렇습니다.<br/>

그리고, 여러가지 과정을 거쳐서 `프론트앤드`로 전향했고, 이를 업으로 삼은 만큼 제가 행복해하고, 재밌어하는 경험을 극대화 시켜서 저의 장기로 내세우고 싶다는 생각이 들었습니다.<br/>

처음부터 리액트를 파봐야겠다고 생각했던게 아니었습니다.<br/>

단순히 이런 경험을 해보고 싶다였으나.. 네이버 부스트캠프에서 리액트를 만들어보거나, 이를 깊게 탐구해보면서 정말 큰 재미를 느꼈고, 이에 따라서 한번 딥다이브를 도전해보게 되었습니다. <br/>
