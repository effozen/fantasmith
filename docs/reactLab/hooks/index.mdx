## 리액트 훅 톺아보기

리액트가 대세가 된 결정적 계기는 훅이었다고 생각합니다.

> "make it easier build great UI"

리액트가 가진, 훌륭한 UI를 위한 쉬운 빌드라는 철학을 바탕으로, DX 측면에서 엄청난 개선을 이룬 시기가 바로 훅이 도입된 이후라고 생각합니다.

이번 카테고리에서는 리액트 훅을 직접 구현해보고, 이를 통해 리액트의 동작 방식을 깊이 있게 이해해보는 시간을 가지고자 합니다.
