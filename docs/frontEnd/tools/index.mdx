## 🚀 프레임워크/라이브러리에 관하여

프레임워크와 라이브러리는 개념적으로는 비슷하지만, 사용하는 방법이 다르다고 할 수 있습니다.

프레임워크는 내가 작성한 코드를 프레임워크가 실행시키는 방식으로 사용하며, 라이브러리는 내가 작성한 코드 안에서 라이브러리를 실행시키는 방식으로 사용합니다.

이러한 도구들은 개발 생산성을 크게 증가시켜주는 역할을 합니다. 어느정도 자동화 도구를 제공해준다거나, 기능들을 미리 구현해서 가져다 쓸 수 있게 해주는 것이지요.

특히나 `React`가 상당히 강조되는 현재의 프론트앤드 개발에서는 더더욱 이런 도구가 중요해지는 듯 합니다.

이 카테고리에서는 이러한 프레임워크와 라이브러리를 어떻게 사용하고, 그 과정에서 마주한 에러 처리와 이해 과정을 담고자 합니다.
